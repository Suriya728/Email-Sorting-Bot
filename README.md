# Project-Email_Sorting_Bot
ระบบคัดแยกอีเมลอัตโนมัติ

# แนวคิดและที่มาของ project
เนื่องจากการได้รับอีเมลจำนวนมาก ทำให้การตรวจสอบและคัดแยกอีเมลด้วยคนใช้เวลานาน และอาจพลาดอีเมลสำคัญ เช่น ใบแจ้งหนี้หรืออีเมลสมัครงาน ดังนั้นproject นี้จึงพัฒนาระบบ RPA เพื่อช่วยคัดแยกอีเมลอัตโนมัติ ลดเวลาในการทำงาน และบันทึกข้อมูลอีเมลลงใน Excel เพื่อให้สามารถจัดการข้อมูลได้อย่างมีประสิทธิภาพ
