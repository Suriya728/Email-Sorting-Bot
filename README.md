# Project-Email_Sorting_Bot
ระบบคัดแยกอีเมลอัตโนมัติ

# แนวคิดและที่มาของ project
เนื่องจากการได้รับอีเมลจำนวนมาก ทำให้การตรวจสอบและคัดแยกอีเมลด้วยคนใช้เวลานาน และอาจพลาดอีเมลสำคัญ เช่น ใบแจ้งหนี้หรืออีเมลสมัครงาน ดังนั้นproject นี้จึงพัฒนาระบบ RPA เพื่อช่วยคัดแยกอีเมลอัตโนมัติ ลดเวลาในการทำงาน และบันทึกข้อมูลอีเมลลงใน Excel เพื่อให้สามารถจัดการข้อมูลได้อย่างมีประสิทธิภาพ

# ขั้นตอนการทำงานของระบบของระบบคัดแยกเมล 


ขั้นตอนที่ 1: การเตรียมพื้นที่เก็บข้อมูล (Build Data Table)
- เมื่อเริ่มรันระบบ บอทจะใช้คำสั่ง Build Data Table 3 ครั้ง เพื่อสร้างตารางข้อมูลจำลองขึ้นมาเตรียมไว้ในระบบ  ตาราง Finance , HR , และ General
- ภายในแต่ละตารางจะมีการกำหนดหัวคอลัมน์ไว้ 3 ช่อง คือ Sender (ผู้ส่ง), Subject (หัวข้อ), และ Date (วันที่) Has Attachment? (มีไฟล์แนบมาด้วยหรือไม่) เพื่อเตรียมรอรับข้อมูลอีเมล

![](Email-Sorting-Bot%20_Github/6.jpg)


ขั้นตอนที่ 2 : การดึงอีเมลเข้าสู่ระบบ (For Each Email)
- บอทใช้คำสั่ง For Each Email เข้าถึงกล่องจดหมาย Inbox เพื่อดึงอีเมลมาประมวลผลทีละฉบับ 
- โดยระบบจะดึงมาเฉพาะอีเมลที่สถานะยังไม่ได้อ่านเท่านั้น (UnreadOnly="True")
